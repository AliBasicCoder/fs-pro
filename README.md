# fs-pro

a library to work with files as objects

see the full docs [here](https://alibasiccoder.github.io/fs-pro/)

## Usage

```js
import { File } from "fs-pro";

const file = new File("hello_world.txt");

file.write("hello").append("world");
// ...
```
